# 📌 Nome del Progetto
Slay the Balrog è un rpg arcade con sistema di combattimento a turni, l'obbiettivo del gioco è battere il nemico più forte del gioc (Il Balrog).
All'avvio si avrà l'opzione di iniziare una nuova partita o di continuarne una esistente.
Il loop principale del gioco consiste nel scegliere il nemico che si vuole combattere, alla fine di ogni combattimento si tornerà al menù principale.
---

## 🚀 Come eseguire il progetto

### Prerequisiti
- Java 25 (LTS)
- Gradle

### Istruzioni

```bash
git clone <https://github.com/Ema1180/rpg-mdp>
cd <rpg-mdp>
```

### Build del progetto
```bash
./gradlew build
```

### Esecuzione
```bash
./gradlew run
```
